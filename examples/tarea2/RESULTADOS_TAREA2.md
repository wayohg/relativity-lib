Por tanto `dγ/dt = 0` y:

```text
F = m γ a
```

## Ejercicio 3

De `E = γE0 = E0 + K` se tiene:

```text
γ = (E0 + K)/E0
```

Como:

```text
γ = 1/sqrt(1 - β²)
```

entonces:

```text
β = sqrt(1 - (E0/(E0 + K))²)
```

## Ejercicio 5

Con `β = 0.6`, `γ = 1.25` y `m_e c² = 0.511 MeV`:

```text
E_fotón ≈ γ m_e c² ≈ 0.63875 MeV
ν ≈ 1.544 × 10^20 Hz
```

No puede formarse un solo fotón porque el momento inicial total es cero,
pero un fotón único con energía no nula necesariamente tiene momento `p = E/c`.
